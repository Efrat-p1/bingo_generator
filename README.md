# מחולל לוחות בינגו תמונות לילדים (Serverless Web App) 🎲

מערכת אינטרנט מלאה (HTML + CSS + JavaScript) ליצירת לוחות בינגו מבוססי תמונות לילדים שאינם יודעים לקרוא.
המערכת פועלת **בצד הלקוח בלבד (Client-Side Only)** ללא צורך בשרת, ומיועדת לפריסה מהירה וחינמית ב-**GitHub Pages**.

---

## 🚀 איך להריץ מקומית?

אין צורך בהתקנת ספריות או הרצת שרת פייתון! 
פשוט פתחו את הקובץ **`index.html`** בדפדפן האינטרנט שלכם (דאבל-קליק על הקובץ).

---

## 🌐 פריסה חינמית ב-GitHub Pages

כדי להעלות את הפרויקט לרשת כאתר אינטרנט ציבורי שנגיש מכל מכשיר:

1. **פתחו רפוזיטורי (Repository) חדש ב-GitHub**:
   - תנו לו שם (למשל: `bingo-generator`).
   - הגדירו אותו כ-**Public**.

2. **העלו את הקבצים הבאים**:
   - `index.html`
   - תיקיית `css/` (המכילה את `style.css`)
   - תיקיית `js/` (המכילה את `balancing.js`, `image_helper.js`, `ppt_builder.js` ו-`app.js`)

3. **הפעילו את GitHub Pages**:
   - היכנסו לכרטיסיית **Settings** (הגדרות) של הרפוזיטורי בתוך גיטהאב.
   - בחרו בתפריט הצדדי ב-**Pages**.
   - תחת **Build and deployment**, בחרו בתוך **Branch** את הענף `main` (או `master`) ולחצו על **Save**.

תוך דקה או שתיים האתר שלכם יהיה באוויר בכתובת:
`https://[username].github.io/[repository-name]/`

---

## 📁 מבנה התיקיות

- `index.html` - המבנה הראשי של ממשק המשתמש והגדרות הספריות מ-CDN.
- `css/style.css` - עיצוב מודרני מבוסס Glassmorphism, תמיכה רספונסיבית לניידים ויישור RTL לעברית.
- `js/balancing.js` - אלגוריתם חלוקת תמונות מאוזנת וייחודית לכל לוח.
- `js/image_helper.js` - כלי לשינוי גודל תמונות תוך שמירה על יחס המימדים המקורי (ללא חיתוך).
- `js/ppt_builder.js` - יצירת מצגת ה-Powerpoint בפורמט A4 לרוחב (שני לוחות A5 בשקף) עם PPTXGenJS.
- `js/app.js` - ניהול אירועי הממשק, טבלאות וסטטיסטיקה מבוססת טבלת HTML/CSS.
